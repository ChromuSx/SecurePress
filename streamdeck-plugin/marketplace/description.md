SecurePress adds Windows Hello protection to Stream Deck actions so sensitive shortcuts only run after explicit user verification. Use one button to launch programs, trigger hotkeys, run PowerShell, Batch, or Python scripts, send HTTP requests, type or paste text, or execute a multi-command sequence after PIN, fingerprint, or face authentication.

Sensitive fields such as script content, HTTP bodies, headers, text input, and command arguments are stored outside Stream Deck profiles with Windows DPAPI protection for the current Windows user. SecurePress also supports short authentication sessions, or requiring Windows Hello every time for stricter workflows.

Requirements: Windows 10 or Windows 11, Stream Deck 6.9 or newer, and Windows Hello configured with PIN, fingerprint, or facial recognition.
